# BaseDare Brain Vault Agent Rules

When working in this vault:

1. Read `00-control/today.md`, `00-control/scorecard.md`, and `00-control/open-loops.md` before proposing growth work.
2. Check existing lead files before creating a new lead.
3. Prefer updating a lead's `next_action`, `status`, `last_touched_at`, and `evidence` over creating duplicate notes.
4. Outreach can be drafted automatically, but messages stay in `02-outreach/drafts/` until reviewed.
5. Log reusable market learnings in `03-insights/` so the next session gets sharper.
6. Track experiments in `04-experiments/` with a hypothesis, metric, status, and result.
7. Never mark an external action as sent unless there is evidence that a human sent it or explicitly approved sending.
8. Keep notes concise, structured, and grounded in evidence.

Safe automatic work:
- research
- list building
- dedupe
- ranking
- enrichment from public or internal context
- drafting
- summarization
- memory updates

Review required:
- outbound email or DM sending
- public posts
- partner proposals
- pricing, perks, or pilot promises
- follow-up sequences

Human only:
- contracts
- payments
- treasury actions
- legal commitments
- deleting important records
- granting credentials
- live production admin decisions
