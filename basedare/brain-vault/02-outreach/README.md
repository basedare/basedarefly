# Outreach

Drafting is safe. Sending is expensive.

Use:

- `drafts/` for messages not yet approved.
- `sent/` only after a human sends or explicitly approves sending.
- `follow-ups/` for due follow-up queues.

Every outreach note should include:

- objective
- audience
- personalization hook
- draft copy
- approval status
- linked lead
- reason it should work
