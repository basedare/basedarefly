# Daily Operator Loop

Use this at the start of any BaseDare Brain session.

## 1. Read Control

- [[00-control/today]]
- [[00-control/scorecard]]
- [[00-control/open-loops]]
- [[05-playbooks/loop-map]]

## 2. Pick The Highest-Leverage Objective

Prefer work that can create:

- a qualified lead
- a better follow-up
- a warm conversation
- a useful experiment
- a reusable learning

## 3. Check Memory

Before creating a new lead or draft:

- search existing lead files
- check sent outreach
- check objections
- check city signals

## 4. Do Safe Work Automatically

Allowed:

- research
- enrichment
- dedupe
- ranking
- draft generation
- memory updates
- scorecard updates

## 5. Package Review Items

Anything external goes into:

- `02-outreach/drafts/`
- `00-control/open-loops.md`

## 6. Close The Loop

Update:

- scorecard
- open loops
- affected lead files
- any new insight or experiment
