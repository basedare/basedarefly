# BaseDare Brain Vault

This is the repo-native memory layer for BaseDare Brain.

Obsidian can open this folder as a vault, but BaseDare does not depend on Obsidian. The durable asset is plain Markdown with structured frontmatter that agents and humans can both read and update.

## Structure

- `00-control/`: daily command surface, scorecards, and unresolved loops.
- `01-leads/`: venue, creator, sponsor, and partner memory.
- `02-outreach/`: drafts, sent-message logs, and follow-up queues.
- `03-insights/`: city signals, copy learnings, objections, and reusable truths.
- `04-experiments/`: growth and product experiments with explicit hypotheses.
- `05-playbooks/`: repeatable operating procedures.
- `templates/`: copyable frontmatter templates for new records.

## Operating Rule

BaseDare Brain may research, dedupe, rank, draft, summarize, and update memory automatically. External sends, public posts, commercial promises, legal commitments, payments, and destructive actions require human review.
